## Project Overview

This portfolio was designed to present my work as a full-stack developer with a strong focus on Java, Spring Boot, React, TypeScript, security, deployment and real business solutions.

## Main Features

- [x] English, Portuguese and Spanish content
- [x] Responsive layout for mobile, tablet and desktop
- [x] Dark and light themes
- [x] Project case studies with live links
- [x] Accessible navigation and semantic page structure
- [x] Static deployment through Cloudflare Pages

## How to Run the Project

### Requirements

- Node.js 20 or later
- npm

### Installation

```bash
git clone https://github.com/brielmarca/gabriel-marca-portfolio.git
cd gabriel-marca-portfolio
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

Create a production build with:

```bash
npm run build
```
